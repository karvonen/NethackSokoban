# NethackSokoban

Java programming project for [Joululabra 2015](https://github.com/javaLabra/Joululabra-2015)

##Movement keys:
###Numpad
    7 8 9
     \|/
    4- -6
     /|\
    1 2 3


###Vi keys with added roguelike diagonals
    y k u
     \|/
    h- -l
     /|\
    b j n
